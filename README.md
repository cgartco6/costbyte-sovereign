# Costbyte Sovereign Agency

Minimal sovereign marketplace + terminal forge. Designed for air-gapped or high-security deployments.

## Structure
- `index.html` — Main marketplace
- `intake.html` — Terminal Forge intake
- `assets/sovereign.css` — Core styling engine
- `core/` — Auth & cart logic (client-side)
- `3794.jpg` — Hero visual asset

## Deployment
1. Clone or download the repo
2. Host statically (GitHub Pages, Netlify, IPFS, or self-hosted Nginx/Caddy)
3. For production: enable HTTPS + Content-Security-Policy
4. CI/CD runs HTML/CSS linting on every push

## Philosophy
Sovereign by default. No telemetry. No third-party dependencies. Pure static + client JS.

Made for operators who value control over convenience.
